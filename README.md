# Contact System

This is an exam.

## Installation

1. Clone the repository to your local machine:

```bash
git clone <repository_url>


2. Navigate to the project directory:
cd <project_directory>

3. Install PHP dependencies using Composer:
composer install

4. Copy the .env.example file and rename it to .env. Update the .env file with your environment configuration, including database connection details.
5. Generate the application key:
php artisan key:generate

6. Run database migrations to create the necessary tables:
php artisan migrate

7. Install Node.js dependencies and compile assets (if applicable):
npm install && npm run dev

## Usage
To start the development server, run:
php artisan serve

Access the application in your web browser at http://localhost:8000.
